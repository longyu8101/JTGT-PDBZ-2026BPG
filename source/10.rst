10 斜拉桥上部结构构件结构性缺损评定
========================================

.. raw:: html

  <h2 id="test111">10 斜拉桥上部结构构件结构性缺损评定</h2>

10.1 一般规定
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id10.1.1"> 10.1.1</a> <span id="id10.1.1"> 斜拉桥上部结构应对主梁、斜拉索、索塔、约束体系等构件进行结构性缺损评定。</span></p>
 <p style="text-align:justify"><a href="#id10.1.2"> 10.1.2</a> <span id="id10.1.2"> 主梁设置分离式桥面板的，桥面板及其支座等构件的缺损计入主梁缺损标度。</span></p>
 
10.2 主梁
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id10.2.1"> 10.2.1</a> <span id="id10.2.1"> 预应力混凝土梁结构性缺损评定应按本标准 <a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.2.11">6.2.11条</a>执行。</span></p>
 <p style="text-align:justify"><a href="#id10.2.2"> 10.2.2</a> <span id="id10.2.2"> 钢箱梁结构性缺损评定应按本标准 <a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.2.12">6.2.12条</a>执行。</span></p>
 <p style="text-align:justify"><a href="#id10.2.3"> 10.2.3</a> <span id="id10.2.3"> 钢桁梁结构性缺损评定应按本标准 <a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.2.13">6.2.13条</a>执行。</span></p>
 <p style="text-align:justify"><a href="#id10.2.4"> 10.2.4</a> <span id="id10.2.4"> 钢—混凝土组合梁结构性缺损评定应按本标准<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.2.15">6.2.15条</a>执行。</span></p>
 <p style="text-align:justify"><a href="#id10.2.5"> 10.2.5</a> <span id="id10.2.5"> 混凝土桥面板结构性缺损评定应按本标准<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.2.1">6.2.1条</a>执行。</span></p>
 <p style="text-align:justify"><a href="#id10.2.6"> 10.2.6</a> <span id="id10.2.6"> 钢桥面板结构性缺损评定应按本标准 <a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.2.14">6.2.14条</a>执行。</span></p>
 <p style="text-align:justify"><a href="#id10.2.7"> 10.2.7</a> <span id="id10.2.7"> 板桁分离式桥面板支座结构性缺损评定应按本标准<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/06.html#id6.5.1">6.5.1 </a> 执行。</span></p>
