  <h2 id="test111">2 术语和符号</h2>


2.1 术语
--------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id2.1.1"> 2.1.1</a> <span id="id2.1.1"> 桥梁技术状况 Bridge Condition<br>
 反映构件、桥跨或桥梁保持其设计使用功能和工作性能的综合技术状态。</span></p>
 <p style="text-align:justify"><a href="#id2.1.2"> 2.1.2</a> <span id="id2.1.2"> 桥梁技术状况等级 Bridge Condition Level<br>
 反映桥跨或全桥技术状况的指标，以 1~5 类表示。</span></p>
 <p style="text-align:justify"><a href="#id2.1.3"> 2.1.3</a> <span id="id2.1.3"> 桥梁技术状况指数 Bridge Condition Index<br>
 反映桥跨或全桥技术状况的指标，以百分制表示。</span></p>
 <p style="text-align:justify"><a href="#id2.1.4"> 2.1.4</a> <span id="id2.1.4"> 桥梁构件 Bridge Element<br>
 在桥梁中独立承担或构成某项功能的最小单元。如一片梁、一个桥墩、一套伸缩装置等。</span></p>
 <p style="text-align:justify"><a href="#id2.1.5"> 2.1.5</a> <span id="id2.1.5"> 桥梁部件 Bridge Component<br>
 桥梁中同类构件的集合。</span></p>
 <p style="text-align:justify"><a href="#id2.1.6"> 2.1.6</a> <span id="id2.1.6"> 专项检查 Special Inspection<br>
 为查明构件缺损程度、缺损发展变化速度或缺损产生原因而开展的深度检查。</span></p>
 <p style="text-align:justify"><a href="#id2.1.7"> 2.1.7</a> <span id="id2.1.7"> 构件技术状况等级 Element Condition Level<br>
 反映构件缺损状况、功能状况、影响状况的综合技术指标，以 1~5 类表示。</span></p>
 <p style="text-align:justify"><a href="#id2.1.8"> 2.1.8</a> <span id="id2.1.8"> 构件缺损状况标度 Element Defective Index<br>
 反映构件现状与原设计状态差异的技术指标，包括耐久性缺损和结构性缺损。</span></p>
 <p style="text-align:justify"><a href="#id2.1.9"> 2.1.9</a> <span id="id2.1.9"> 构件耐久性缺损标度 Element Durability Defect Index<br>
 反应构件初始缺陷、材质劣化、防护措施失效等维持设计使用年限或目标使用年限降低的指标。</span></p>
 <p style="text-align:justify"><a href="#id2.1.10"> 2.1.10</a> <span id="id2.1.10"> 构件结构性缺损标度 Element Structural Defect Index<br>
 反应构件开裂、变形、变位、组件失效等维持设计或目标使用力学性能降低的指标。</span></p>
 <p style="text-align:justify"><a href="#id2.1.11"> 2.1.11</a> <span id="id2.1.11"> 构件功能状况标度 Element Function Index<br>
 反映构件设计使用功能异常程度的技术指标。</span></p>
 <p style="text-align:justify"><a href="#id2.1.12"> 2.1.12</a> <span id="id2.1.12"> 构件影响状况标度 Element Affected Index <br>
 反映构件缺损发展趋势、缺损或功能异常对相关构件、桥跨或全桥安全影响程度的技术指标。</span></p>
