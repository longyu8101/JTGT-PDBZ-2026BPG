14 构件功能状况评定
========================================

.. raw:: html

  <h2 id="test111">14 构件功能状况评定</h2>

14.1 一般规定
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id14.1.1"> 14.1.1</a> <span id="id14.1.1"> 构件功能状况评定应综合考虑构件的使用功能、安全性和耐久性等因素。</span></p>
 <p style="text-align:justify"><a href="#id14.1.2"> 14.1.2</a> <span id="id14.1.2"> 本标准未列出的功能或构件应根据构件自身功能特性，应按本标准<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/03.html#id3.2.2">3.2.2 条</a>确定评定标度并说明制定依据。</span></p>
 <p style="text-align:justify;" >条文说明：<br/>本标准列出常用桥梁构件的功能状况评定标度，其他特殊情况需要具备的功能或本标准中未列出的特殊构件，参照<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/03.html#B3.2.3">表 3.2.3 </a>的原则单独制定功能状况评定标度，并在检查评定报告中对制定依据予以说明。</p>
 <p style="text-align:justify"><a href="#id14.1.3"> 14.1.3</a> <span id="id14.1.3"> 构件发生明显异常时，可按本标准<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/FLA.html">附录 A</a>开展专项检查论证构件功能异常程度。</span></p>

----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id14.2.1"> 14.2.1</a> <span id="id14.2.1"> 桥面系功能状况应评定下列内容。</span></p>
 <ol>
 <li>桥面铺装的防排水功能和通过能力。</li>
 <li>伸缩装置的承担功能、锚固性能和工作性能。</li>
 <li>护栏的防撞功能。</li>
 </ol>

14.3 上部结构
---------------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id14.3.1"> 14.3.1</a> <span id="id14.3.1"> 上部结构功能状况应评定下列内容。</span></p>

 <ol>
 <li>构件的承载能力、传力功能、适应温度变形功能和工作性能。</li>
 <li>铰缝及横向联系的传力功能和工作性能。</li>
 <li>拉吊索结构的工作性能。</li>
 <li>支座的传力功能、变形功能和工作性能。</li>
 </ol>

 <p style="text-align:justify;" >条文说明：<br/>承载能力指主要承重构件是否具备设计或通行需求的承载能力。当从缺损状况判定承载能力可能明显降低时，需通过承载能力评定确认其承载能力不足程度；当设计承载能力小于当前通行需求时，需要通过检算或承载能力评定确认其承载能力相对当前通行需求的不足程度。<br/> 适应温度变形功能指主要承重构件由于季节性温差导致的热胀冷缩是否受到限制。如梁端与桥台背墙抵死、杂物阻塞导致无法伸长、支座设置不当或失效导致位移受限等均影响其适应温度变形功能。<br/> 工作性能指主要承重构件的正常变形、变位等是否受限。如主梁异常下挠、异变位、异常振动或产生异响，或其他原因导致的受力体系发生改变等。</p>

 <p style="text-align:justify;" >条文说明：<br/>传力功能指支座是否处于正常传递荷载的状态。支座脱空或失效会导致主要承重构件荷载传递的重分配，造成其他支座承载过大。<br/> 变形功能指板式橡胶支座、隔震橡胶支座等是否能通过正常剪切变形为主要承重构件提供温度变形需要。如老化、剪切变形过大等缺损导致变形功能降低、失效，或支座高度设置过低导致变形功能不足。 <br/>工作性能指滑动（滚动、摆动等）支座是否具有满足主要承重构件正常温度变形需要，提供的位移量是否足够或滑动（滚动、摆动等）状态是否正常。如滑动支座滑动组件失效或未按标准要求设置滑动支座，设置的支座位移量不足或位移超限但未采取有效措施，支座组件损坏或安装组件未拆除导致有效位移不足或失效等。</p>
