7 拱桥上部结构构件结构性缺损评定
=======================================

.. raw:: html

  <h2 id="test111">7 拱桥上部结构构件结构性缺损评定</h2>

7.1 一般规定
---------------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id7.1.1"> 7.1.1</a> <span id="id7.1.1"> 拱桥上部结构应对主拱圈、腹拱墩和侧墙、腹拱圈或桥面板、横向联系等构件进行结构性缺损评定。</span></p>
 <p style="text-align:justify"><a href="#id7.1.2"> 7.1.2</a> <span id="id7.1.2"> 桥面板设置横隔板、铰缝、支座的，横隔板、铰缝、支座缺损应计入桥面板缺损标度。</span></p>
 <p style="text-align:justify"><a href="#id7.1.3"> 7.1.3</a> <span id="id7.1.3"> 桁架拱桥拱片端部设有支座的，支座缺损计入桁架拱片缺损标度。</span></p>
 <p style="text-align:justify"><a href="#id7.1.4"> 7.1.4</a> <span id="id7.1.4"> 刚架拱桥拱片端部设有支座的，支座缺损计入刚架拱片缺损标度。</span></p>
