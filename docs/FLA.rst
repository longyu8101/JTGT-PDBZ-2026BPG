附录A 专项检查
===================================

.. raw:: html

  <h2 id="test111">附录A 专项检查</h2>

.. raw:: html

 <p style="text-align:justify"><a href="#idA.0.1"> A.0.1</a> <span id="idA.0.1">隐蔽构件可能有明显缺损或功能异常时，应采用可靠方案局部拆除防护构造或遮挡物，明确缺损状况、功能状况及影响状况，提出处治建议，处置完成后恢复防护构造或遮挡物。</span></p>
 <p style="text-align:justify"><a href="#idA.0.2"> A.0.2</a> <span id="idA.0.2">构件疑存在内部缺损影响正常功能时，应优先采用无破损检测技术查明原因和程度，无破损检测实施困难时，需经过可靠论证采用局部破损方法检测，查明原因和程度并提出处治建议。</span></p>
 <p style="text-align:justify"><a href="#idA.0.3"> A.0.3</a> <span id="idA.0.3">桥面排水不畅，有明显积水时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 测量桥面线形、桥面纵、横坡设置情况，积水区域加密测点。</li>
 <li> 检查泄水孔位置及排水是否通畅。</li>
 <li> 分析积水原因，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.4"> A.0.4</a> <span id="idA.0.4">台后明显跳车时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 测量桥路衔接处 50 m 范围纵断面，分析桥路高差状况。</li>
 <li> 核查竣工资料，明确台后搭板设置情况。</li>
 <li> 必要时对台后路基进行勘探，掌握路基填筑状况、含水量等。</li>
 <li> 分析跳车产生原因，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.5"> A.0.5</a> <span id="idA.0.5">混凝土构件控制截面裂缝宽度超限，或在活载作用下裂缝处于开合状态时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 绘制裂缝展开图，标明裂缝位置、走向和宽度，分析裂缝内外（或左右）侧的对应关系。</li>
 <li> 与历次检查成果对比分析，明确裂缝发展规律。</li>
 <li> 测量主要裂缝深度，必要时局部钻孔或取芯，掌握裂缝在混凝土内部的扩展方向。</li>
 <li> 分析裂缝成因，必要时结合结构检算，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.6"> A.0.6</a> <span id="idA.0.6">混凝土构件因环境因素、渗漏水长期侵蚀等情况，内部钢筋锈蚀发生可能性较高时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 根据锈蚀发生的条件，选择适当测区。</li>
 <li> 采用半电池电位法测量钢筋锈蚀电位。</li>
 <li> 绘制等点位图，锈蚀显著时扩大测区。</li>
 <li> 分析锈蚀成因，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.7"> A.0.7</a> <span id="idA.0.7">预应力混凝土构件存在顺钢束开裂，且伴有渗水泛碱现象时，宜开展注浆密实度检测，并提出处治建议。</span></p>
 <p style="text-align:justify"><a href="#idA.0.8"> A.0.8</a> <span id="idA.0.8">预应力混凝土构件有明显下挠、开裂，为判断产生原因时，宜采用横张法或其他可靠方法开展有效预应力检测并提出处治建议。</span></p>
 <p style="text-align:justify"><a href="#idA.0.9"> A.0.9</a> <span id="idA.0.9">上部结构主要承重构件承载能力、铰缝及横向联系传力功能明显不足时，应按《公路桥梁承载能力检测评定规程》（JTG/T J21）或《公路桥梁荷载试验规程》（JTG/T J21-01）规定开展检测评定，并提出处治建议。</span></p>
 <p style="text-align:justify"><a href="#idA.0.10"> A.0.10</a> <span id="idA.0.10">主拱圈有横向贯通开裂、拱圈明显变形或拱脚明显位移时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 测量主拱圈及拱上结构尺寸，计算各部分恒载分布。</li>
 <li> 调查桥梁通行荷载情况。</li>
 <li> 测量拱轴线及拱脚位移值。</li>
 <li> 检算主拱圈内力分布，判定桥梁承载能力，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.11"> A.0.11</a> <span id="idA.0.11">加固构造与原结构剥离、钢构件严重锈蚀等导致加固构造与原结构不能共同受力，或预应力组件变形或预应力明显损失导致加固效果下降时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 详细检查加固构造的缺损范围。</li>
 <li> 结合设计文件检算加固构造的有效性。</li>
 <li> 必要时针对加固构造开展荷载试验。</li>
 <li> 提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.12"> A.0.12</a> <span id="idA.0.12">拉吊索内层防护破损、开裂，索体有锈蚀发生可能时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 局部拆除失效部位防护层，检查锈蚀情况。</li>
 <li> 局部拆除索体底端防护层，检查锈蚀情况。</li>
 <li> 拆除锚头防护盖，检查渗漏水及锈蚀情况。</li>
 <li> 分析拉吊索缺损范围及程度，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.13"> A.0.13</a> <span id="idA.0.13">拉吊索索力偏差大于 10%时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 测试全桥索力分布。</li>
 <li> 测试桥面线形。</li>
 <li> 检算全桥拉索安全系数。</li>
 <li> 评估索力重分布对主梁受力的影响，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.14"> A.0.14</a> <span id="idA.0.14">支座系统产生严重剪切变形或滑动时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 根据支座缺损情况逐一测量全跨、全联或全桥支座变形或位移量。</li>
 <li> 绘制支座变形或位移图，判断梁体位移趋势。</li>
 <li> 根据梁体位移趋势检查梁端位移、固定支座缺损情况。</li>
 <li> 检测固定支座所在桥墩变形和裂缝分布。</li>
 <li> 分析病害产生原因及影响程度，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.15"> A.0.15</a> <span id="idA.0.15">桥墩或桥台台身产生倾斜、变位、水平向开裂时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 按照《公路桥梁承载能力检测评定规程》（JTG/T J21）要求开展桥梁恒载变异状况调查。</li>
 <li> 材料参数、截面参数调查。</li>
 <li> 变形、变位状况调查。</li>
 <li> 结构检算判定承载能力，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.16"> A.0.16</a> <span id="idA.0.16">系梁与桩、柱连接处两端出现规则竖向裂缝时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 详细检查开裂情况。</li>
 <li> 测量墩台各节段倾斜、变形情况，必要时局部开挖埋置部分。</li>
 <li> 分析病害产生原因及影响程度，提出处治建议。</li>
 </ol>
 <p style="text-align:justify"><a href="#idA.0.17"> A.0.17</a> <span id="idA.0.17">扩大基础埋深不明或显著冲刷时，宜采用钻孔取芯法或其他可靠方法探明基底位置，依次检算地基承载力并提出处治建议。</span></p>
 <p style="text-align:justify"><a href="#idA.0.18"> A.0.18</a> <span id="idA.0.18">桩基础桩头外露明显时，应查阅基础类型和埋置深度，并与设计局部冲刷深度对比分析，评估桩基的竖向承载力和抗弯承载力。缺乏设计资料时，经勘探确认地质条件及基础埋深，依此检算基桩承载力并提出处治建议。</span></p>
 <p style="text-align:justify"><a href="#idA.0.19"> A.0.19</a> <span id="idA.0.19">桩基础外露长度过大时，应结合原设计计算结果判断桥墩稳定性，无相关资料时应检算桥墩承载力和稳定性。</span></p>
 <p style="text-align:justify"><a href="#idA.0.20"> A.0.20</a> <span id="idA.0.20">河床断面冲刷下切严重或主河槽明显变迁时，应按下列规定开展专项检查：</span></p>
 <ol>
 <li> 测试桥位上下游桥侧河床断面。</li>
 <li> 测试桥位上下游 50 m、100 m 河床断面。</li>
 <li> 结合历次测试报告明确发展规律。</li>
 <li> 结合设计文件分析河床断面冲刷对结构的影响。</li>
 <li> 提出处治建议。</li>
 </ol>
