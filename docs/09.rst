9 悬索桥上部结构构件结构性缺损评定
========================================

.. raw:: html

  <h2 id="test111">9 悬索桥上部结构构件结构性缺损评定</h2>

9.1 一般规定
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id9.1.1"> 9.1.1</a> <span id="id9.1.1"> 悬索桥上部结构应对加劲梁、吊索及索夹、主缆、索塔及主索鞍、自锚式悬索桥锚固系统、约束体系等构件进行结构性缺损评定。</span></p>
 <p style="text-align:justify"><a href="#id9.1.2"> 9.1.2</a> <span id="id9.1.2"> 加劲梁设置分离式桥面板的，桥面板及其支座等构件的缺损计入加劲梁缺损标度。</span></p>
 <p style="text-align:justify"><a href="#id9.1.3"> 9.1.3</a> <span id="id9.1.3"> 吊索和索夹应作为一个构件参与评定。</span></p>
 <p style="text-align:justify;" >条文说明：<br/>《公路悬索桥设计规范》（JTG/T D65-05—2015）规定，吊索与主缆的连接可采用骑跨式或销接式，吊索与加劲梁的连接可采用错头承压式或销接式。由于索夹、吊索和连接构造作为一个整体提供了主缆和加劲梁之间的传力功能，因此作为一个构件的不同部分参与评定。</p>
 <p style="text-align:justify"><a href="#id9.1.4"> 9.1.4</a> <span id="id9.1.4"> 索塔和主索鞍应作为一个构件参与评定。</span></p>
 <p style="text-align:justify;" >条文说明：<br/>索塔和主索鞍作为一个整体为主缆提供支撑，根据《公路悬索桥设计规范》（JTG/T D65-05—2015），主索鞍的顶推位移只是施工期间的一个过程，成桥后应将主索鞍与索塔固结定位，因此作为一个构件的不同部分参与评定。</p>

9.2 加劲梁
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id9.2.1"> 9.2.1</a> <span id="id9.2.1"> 预应力混凝土梁结构性缺损评定应按本标准 6.2.11 条执行。</span></p>
 <p style="text-align:justify"><a href="#id9.2.2"> 9.2.2</a> <span id="id9.2.2"> 钢箱梁结构性缺损评定应按本标准 6.2.12 条执行。</span></p>
 <p style="text-align:justify"><a href="#id9.2.3"> 9.2.3</a> <span id="id9.2.3"> 钢桁梁结构性缺损评定应按本标准 6.2.13 条执行。</span></p>
 <p style="text-align:justify"><a href="#id9.2.4"> 9.2.4</a> <span id="id9.2.4"> 钢板梁结构性缺损评定应按本标准 6.2.14 条执行。</span></p>
 <p style="text-align:justify"><a href="#id9.2.5"> 9.2.5</a> <span id="id9.2.5"> 加钢—混凝土组合梁结构性缺损评定应按本标准 6.2.15 条执行。</span></p>
 <p style="text-align:justify"><a href="#id9.2.6"> 9.2.6</a> <span id="id9.2.6"> 混凝土桥面板结构性缺损评定应按本标准 6.2.1 条执行。</span></p>
