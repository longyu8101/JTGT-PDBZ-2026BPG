15 构件影响状况评定
========================================

.. raw:: html

  <h2 id="test111">15 构件影响状况评定</h2>

15.1 一般规定
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id15.1.1"> 15.1.1</a> <span id="id15.1.1"> 上、下部结构主要构件存在下列情况应开展影响状况评定：</span></p>

 <ol>
 <li> 缺损状况标度为 3。</li>
 <li> 功能状况标度为 2。</li>
 <li> 附着其他设施。</li>
 <li> 桥下空间占用。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.1.2"> 15.1.2</a> <span id="id15.1.2"> 构件影响状况评定应按照缺损发展影响、缺损致灾可能、功能丧失致灾可能和发生灾害影响等方面按照<a href="#B15.1.2">表 15.1.2</a> 执行。</span></p>

 <p style="text-align:justify"><a href="#id15.1.3"> 15.1.3</a> <span id="id15.1.3"> 评定构件缺损发展稳定的情形应经过可靠论证，明确缺损的诱发因素并确保长期稳定方可判定，并在后期持续跟踪缺损的发展变化。</span></p>
 <p style="text-align:justify"><a href="#id15.1.4"> 15.1.4</a> <span id="id15.1.4"> 缺损致灾可能和功能丧失致灾可能宜按<a href="#id3"> 15.2 </a>和 <a href="#id4"> 15.3 </a>评定。必要时可参考附录 A 开展专项检查论证缺损或功能异常对结构安全运行的影响。</span></p>
 <p style="text-align:justify"><a href="#id15.1.5"> 15.1.5</a> <span id="id15.1.5"> 本标准未列出的其他影响因素或构件，可按照<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/03.html#B3.2.4">表 3.2.4</a> 制定相应评定标度并说明制定依据。</span></p>
 <p style="text-align:justify;" >条文说明：<br/>本标准列出常用桥梁构件的影响状况评定标度，其他特殊情况可能造成的影响或本标准中未列出的特殊构件，参照本标准<a href="https://jtgt-pdbz-2026bpg.readthedocs.io/zh-cn/latest/03.html#id3.2.4">3.2.4 条</a>单独制定影响状况评定标度，并在检查评定报告中对制定依据予以说明。</p>

15.2 伸缩缝装置
----------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id15.2.1"> 15.2.1</a> <span id="id15.2.1"> 整体式梁桥影响状况应评定下列内容：</span></p>
 <ol>
 <li> 主梁的缺损、变形、位移及其影响。</li>
 <li> 支座的缺损、位移及其影响。</li>
 <li> 牛腿的缺损及其影响。</li>
 <li> 独柱墩桥梁抗倾覆能力分析结果。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.2.2"> 15.2.2</a> <span id="id15.2.2"> 装配式梁桥影响状况应评定下列内容：</span></p>
 <ol>
 <li> 主梁的缺损、变形、位移及其影响。</li>
 <li> 横向联系的缺损、变形及其影响。</li>
 <li> 牛腿的缺损及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.2.3"> 15.2.3</a> <span id="id15.2.3"> 拱桥影响状况应评定下列内容：</span></p>
 <ol>
 <li> 主拱圈、腹拱圈的缺损、变形、拱脚位移及其影响。</li>
 <li> 腹拱墩的缺损、倾斜及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.2.4"> 15.2.4</a> <span id="id15.2.4"> 组合体系拱桥影响状况应评定下列内容：</span></p>
 <ol>
 <li> 主梁、桥面板的缺损、变形、位移及其影响。</li>
 <li> 系杆的缺损、变形及其影响。</li>
 <li> 横梁的缺损、变形及其影响。</li>
 <li> 吊索的缺损、索力分配、冗余度及其影响。</li>
 <li> 立柱的缺损、倾斜及其影响。</li>
 <li> 主拱圈的缺损、变形、拱脚位移及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.2.5"> 15.2.5</a> <span id="id15.2.5"> 悬索桥影响状况应评定下列内容：</span></p>
 <ol>
 <li> 加劲梁的缺损、变形、位移及其影响。</li>
 <li> 吊索的缺损、索力分配及其影响。</li>
 <li> 主缆的缺损及其影响。</li>
 <li> 索塔及索鞍的缺损及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.2.6"> 15.2.6</a> <span id="id15.2.6"> 斜拉桥影响状况应评定下列内容：</span></p>
 <ol>
 <li> 主梁的缺损、变形、位移及其影响。</li>
 <li> 斜拉索的缺损、索力分配及其影响。</li>
 <li> 索塔的缺损及其影响。</li>
 </ol> 
 
15.3 下部结构
---------------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id15.3.1"> 15.3.1</a> <span id="id15.3.1"> 桥墩影响状况应评定下列内容：</span></p>
 <ol>
 <li> 桥墩的缺损、倾斜及其影响。</li>
 <li> 盖梁、墩帽的缺损及其影响。</li>
 <li> 系梁的缺损及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.3.2"> 15.3.2</a> <span id="id15.3.2"> 桥台影响状况应评定下列内容：</span></p>
 <ol>
 <li> 桥台的缺损、倾斜、变位及其影响。</li>
 <li> 盖梁、台帽的缺损及其影响。</li>
 <li> 系梁的缺损及其影响。</li>
 <li> 防护设施、台后填土状况及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.3.3"> 15.3.3</a> <span id="id15.3.3"> 扩大基础影响状况应评定下列内容：</span></p>
 <ol>
 <li> 基础的缺损及其影响。</li>
 <li> 基础的冲刷、变位及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.3.4"> 15.3.4</a> <span id="id15.3.4"> 桩基础影响状况应评定下列内容：</span></p>
 <ol>
 <li> 基础的缺损及其影响。</li>
 <li> 基础的冲刷及其影响。</li>
 </ol>
 <p style="text-align:justify"><a href="#id15.3.5"> 15.3.5</a> <span id="id15.3.5"> 沉井基础影响状况应评定下列内容：</span></p>
 <ol>
 <li> 基础的缺损及其影响。</li>
 <li> 基础的冲刷及其影响。</li>
 </ol>
