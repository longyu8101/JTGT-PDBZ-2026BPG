2 术语和符号

2.1 术语

2.1.1 桥梁技术状况 Bridge Condition
反映构件、桥跨或桥梁保持其设计使用功能和工作性能的综合技术状态。

2.1.2 桥梁技术状况等级 Bridge Condition Level
反映桥跨或全桥技术状况的指标，以 1~5 类表示。

2.1.3 桥梁技术状况指数 Bridge Condition Index
反映桥跨或全桥技术状况的指标，以百分制表示。

2.1.4 桥梁构件 Bridge Element
在桥梁中独立承担或构成某项功能的最小单元。如一片梁、一个桥墩、一套伸缩装置等。

2.1.5 桥梁部件 Bridge Component
桥梁中同类构件的集合。

2.1.6 专项检查 Special Inspection
为查明构件缺损程度、缺损发展变化速度或缺损产生原因而开展的深度检查。

2.1.7 构件技术状况等级 Element Condition Level
反映构件缺损状况、功能状况、影响状况的综合技术指标，以 1~5 类表示。

2.1.8 构件缺损状况标度 Element Defective Index
反映构件现状与原设计状态差异的技术指标，包括耐久性缺损和结构性缺损。

2.1.9 构件耐久性缺损标度 Element Durability Defect Index
反应构件初始缺陷、材质劣化、防护措施失效等维持设计使用年限或目标使用年限降低的指标。

2.1.10 构件结构性缺损标度 Element Structural Defect Index
反应构件开裂、变形、变位、组件失效等维持设计或目标使用力学性能降低的指标。

2.1.11 构件功能状况标度 Element Function Index
反映构件设计使用功能异常程度的技术指标。

2.1.12 构件影响状况标度 Element Affected Index
反映构件缺损发展趋势、缺损或功能异常对相关构件、桥跨或全桥安全影响程度的技术指标。

2.2 符号

ACCIi —— 第 i 跨附属构造技术状况指数；
BCI —— 桥梁技术状况指数；
BCL —— 桥梁技术状况等级；
BSCIi —— 第 i 跨桥梁技术状况指数；
BSCLi —— 第 i 跨桥梁技术状况等级；
CIi —— 第 i 跨某部件技术状况统计值；
CIav —— 全桥某部件所有同类构件技术状况标度平均值；
CIcirt(i) —— 第 i 跨某部件的构件技术状况标度最差值；
ECL —— 构件技术状况等级；
EDI —— 构件缺损状况标度；
EFI —— 构件功能状况标度；
EAI —— 构件影响状况标度；
FSCIi —— 第 i 跨桥面系技术状况指数；
Li —— 第 i 跨桥梁跨径（m）；
SBCIi —— 第 i 跨下部结构技术状况指数；
SPCIi —— 第 i 跨上部结构技术状况指数；
m —— 桥跨数量；
n —— 桥面系、上部结构、下部结构或附属构造的组成部件数量；
ω —— 构件权重值；
γ —— 桥面系、上部结构、下部结构和附属构造的影响系数。
